## 2. Prompt template to create a product description with the product details:
```
คุณคือผู้เชี่ยวชาญด้าน Social Media Marketing และ Copywriting สำหรับธุรกิจ E-commerce

สร้างโพสต์ Facebook สำหรับขายสินค้า "มือถือ / Smartphone" ที่น่าสนใจ อ่านง่าย และช่วยเพิ่มยอดขาย

เงื่อนไข:
- ภาษา: ไทย
- โทน: [เป็นกันเอง / พรีเมียม / สายเทคโนโลยี / กระตุ้นการขาย]
- กลุ่มเป้าหมาย: [นักเรียน / Gamer / คนทำงาน / Creator / ผู้ใช้งานทั่วไป]
- ความยาว: กระชับ อ่านง่าย
- เน้น "ประโยชน์ที่ลูกค้าได้รับ" มากกว่าการบอกสเปกอย่างเดียว
- ใช้ Emoji ได้อย่างเหมาะสม
- มี Hook เปิดโพสต์ให้น่าสนใจ
- มี Selling Points
- มี Promotion
- มี Call To Action (CTA)
- ปิดท้ายด้วย Hashtags

ข้อมูลสินค้า:

แบรนด์: [ชื่อแบรนด์]

รุ่น: [ชื่อรุ่น]

ราคา: [ราคา]

คุณสมบัติเด่น:
- [Feature 1]
- [Feature 2]
- [Feature 3]
- [Feature 4]

จุดขาย (Selling Points):
- [จุดขาย 1]
- [จุดขาย 2]

โปรโมชั่น:
- ส่วนลด: [XX% OFF]
- ของแถม: [ของแถม]
- ผ่อน: [รายละเอียดผ่อน]
- จัดส่งฟรี: [มี/ไม่มี]

รูปแบบผลลัพธ์:

1. Headline / Hook เปิดโพสต์
2. คำอธิบายสินค้าแบบชวนซื้อ
3. Selling Points แบบ Bullet
4. โปรโมชั่น
5. CTA กระตุ้นการสั่งซื้อ
6. Hashtags
7. ห้ามใส่ข้อมูลที่ไม่ได้รับมา และ emoticon ต่าง ๆ
```
